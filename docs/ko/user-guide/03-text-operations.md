# Part 3: 텍스트 선택, 복사 & 붙여넣기

## 텍스트 선택

### 선택 모드

| 동작 | 결과 |
|--------|--------|
| 클릭 + 드래그 | 일반 텍스트 선택 |
| `Alt` + 클릭 + 드래그 | 블록(사각형) 선택 |
| 더블클릭 | 단어 선택 |
| 트리플 클릭 | 전체 줄 선택 |

### 키보드 선택

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+Shift+A` | 모든 텍스트 선택 |
| `Shift+Arrows` | 문자 단위로 선택 영역 확장 |
| `Ctrl+Shift+Left/Right` | 단어 단위로 선택 영역 확장 |

### 선택 영역 하이라이트

선택된 텍스트는 반투명 파란색 오버레이로 하이라이트돼요. 블록 선택의 경우 행별로 사각형이 렌더링돼요.

### 선택 영역 단어 수

텍스트가 선택되면 상태 표시줄에 문자 수와 단어 수가 표시돼요:
- `SEL:42c/7w` — 42문자, 7단어
- 단어가 0개인 경우 `w` 접미사가 생략돼요 (공백만 있는 선택)

## 복사

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+Shift+C` | 선택 영역을 클립보드에 복사 |
| `Ctrl+Insert` | 선택 영역 복사 (Linux/Windows 관행) |
| `Ctrl+Shift+Alt+H` | 선택 영역을 HTML로 복사 (색상 포함) |
| `Ctrl+Shift+Alt+O` | 마지막 명령 출력 복사 (OSC 133 마크 사용) |
| `Ctrl+Shift+Alt+P` | 현재 작업 디렉토리 경로 복사 |

Command Palette를 통한 추가 복사 명령:
- **보이는 텍스트 복사** — 화면에 보이는 부분만 복사 (스크롤백 제외)
- **Markdown으로 복사** — 터미널 출력을 Markdown 형식으로 변환
- **HTML로 복사** — 색상과 서식 유지

### 스마트 복사 다듬기

복사된 텍스트에서 앞뒤 빈 줄이 자동으로 제거돼요.

## 붙여넣기

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+Shift+V` | 클립보드에서 붙여넣기 |
| `Shift+Insert` | 붙여넣기 (Linux/Windows 관행) |
| 중간 클릭 | 선택 영역 붙여넣기 (X11 방식) |

### Bracketed Paste

shell이 bracketed paste를 지원할 때 (대부분의 최신 shell이 지원):
- 붙여넣은 내용이 `ESC[200~ ... ESC[201~`로 감싸져요
- shell이 여러 줄 붙여넣기를 안전하게 처리할 수 있어요

### 안전한 붙여넣기

bracketed paste를 지원하지 않을 때:
- 후행 newline이 제거되어 실수로 명령이 실행되는 것을 방지해요
- Toast 알림: "첫 줄 붙여넣음 (N줄 제거됨)"

## 검색

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+Shift+F` | 플로팅 검색 바 토글 |
| `Enter` | 다음 일치 항목 |
| `Shift+Enter` | 이전 일치 항목 |
| `Tab` (검색 중) | 대소문자 구분 토글 |
| `Shift+Tab` (검색 중) | 정규식 검색 모드 토글 |
| `Up/Down` (검색 중) | 검색 히스토리 탐색 |
| `Esc` | 검색 바 닫기 |

### 검색 기능

- **플로팅 검색 바**에 일치 항목 카운터 표시 (예: "3/12 일치")
- **검색 히스토리**: 최근 20개 쿼리 저장, Up/Down으로 탐색
- **대소문자 토글**: 대소문자 구분/무시
- **정규식 모드**: 완전한 정규식 지원
- **하이라이트**: 일치 항목이 터미널 grid에 하이라이트돼요

Command Palette를 통한 추가 검색 명령:
- **선택 영역 검색** — 현재 선택된 텍스트로 검색
- **웹에서 검색** — 선택된 텍스트를 브라우저에서 검색

## 내보내기

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+Shift+Alt+S` | 스크롤백을 텍스트 파일로 저장 (`~/ggterm-export-{timestamp}.txt`) |
| `Ctrl+Shift+Alt+E` | 터미널을 HTML로 내보내기 (색상 포함) |

Command Palette를 통한 내보내기 명령:
- **스크롤백 내보내기** — 일반 텍스트 파일
- **HTML로 내보내기** — 색상, 서식, 하이퍼링크 유지

## 터미널 잠금

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+Shift+Alt+L` | 터미널 잠금 토글 (읽기 전용 모드) |

잠겨 있으면 모든 키보드 입력이 차단돼요. 출력을 읽는 동안 실수로 입력하는 것을 방지할 때 유용해요.

## 스크롤

### 키보드 스크롤

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+Shift+Space` | 스크롤백 탐색 모드 토글 (less 방식: j/k/G/g/d/u/q) |
| `Shift+PageUp` | 한 페이지 위로 스크롤 |
| `Shift+PageDown` | 한 페이지 아래로 스크롤 |
| `Shift+Home` | 스크롤백 맨 위로 |
| `Shift+End` | 맨 아래로 스크롤 |
| `Ctrl+Shift+End` | 맨 아래로 스크롤 (대체) |
| `Ctrl+Shift+Alt+Up` | 마크로 스크롤 (OSC 1337 SetMark) |

### 마우스 스크롤

- **스크롤 휠**: 스크롤백 히스토리 스크롤
- **Shift+스크롤**: 모든 pane 동시 스크롤
- **스크롤바**: 오른쪽 가장자리의 얇은 스크롤바를 클릭하거나 드래그

### 스크롤 표시

- **맨 아래로 스크롤 알약**: 위로 스크롤했을 때 아래 화살표가 있는 파란색 알약이 나타나요; 클릭하면 맨 아래로 이동
- **백분율 표시**: 스크롤 위치를 백분율로 표시 (예: "45%")
- **줄 수**: 99줄 이상 스크롤했을 때 줄 수 표시 (예: "127 lines")

### 부드러운 관성 스크롤

트랙패드 모멘텀이 지원되며 지수 감소 보간으로 부드러운 스크롤이 구현돼요.

### Alternate Screen 스크롤

마우스 추적이 없는 전체 화면 앱(vim, less, htop)에서:
- 마우스 휠이 자동으로 화살표 키 입력으로 변환돼요 (DECSET 7727)
- Shift+휠은 이를 우회하고 viewport를 스크롤해요
