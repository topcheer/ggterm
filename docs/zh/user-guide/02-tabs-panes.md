# 第二部分：标签页、窗格与分屏

## 标签页

### 标签页管理

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+T` | 新建标签页 |
| `Ctrl+W` | 关闭当前标签页 |
| `Alt+1-9` | 切换到第 N 个标签页 |
| `Ctrl+Tab` | 下一个标签页 |
| `Ctrl+Shift+Tab` | 上一个标签页 |
| `Ctrl+Shift+\`` | 切换最近标签页（在两个最近的标签页间切换） |
| `Ctrl+Shift+T` | 重新打开上次关闭的标签页 |
| `Ctrl+Shift+N` | 新建窗口 |
| `Ctrl+Shift+I` | 重命名当前标签页 |
| `Ctrl+Shift+PageUp` | 标签页左移 |
| `Ctrl+Shift+PageDown` | 标签页右移 |
| `Ctrl+Shift+Alt+D` | 复制当前标签页（相同 shell + cwd） |
| `Ctrl+Shift+Alt+W` | 关闭所有其他标签页 |

### 标签页交互

- **点击标签页**：切换到该标签页
- **双击标签页**：重命名
- **中键点击标签页**：关闭（浏览器风格）
- **拖拽标签页**：在同级间重新排序
- **右键点击标签页**：上下文菜单（关闭、关闭其他、关闭右侧、固定/取消固定、分屏）
- **点击 "+"**：下拉菜单（新建标签页、水平分屏、垂直分屏）

### 标签页固定

通过命令面板固定标签页以防止意外关闭：
- 固定的标签页显示固定指示器
- 固定标签页上 `Ctrl+W` 被忽略
- 通过命令面板取消固定后方可关闭

### 标签页标题同步

标签页标题自动与运行中的程序同步：
- 显示 OSC 0/2 的程序名称（如 "vim"、"htop"、"less"）
- 回退到 shell 名称（如 "zsh"、"bash"）
- 后台标签页收到 bell 时显示 bell 指示器
- 在备用屏幕模式下显示 `(alt)`

## 分屏窗格

### 创建分屏

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+D` | 水平分屏（左 | 右） |
| `Ctrl+Shift+\` | 垂直分屏（上 / 下） |

新窗格继承当前窗格的工作目录（来自 OSC 7）。

### 窗格导航

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+[` | 聚焦上一个窗格 |
| `Ctrl+Shift+]` | 聚焦下一个窗格 |
| `Alt+H` | 聚焦左侧窗格（vim 风格） |
| `Alt+J` | 聚焦下方窗格（vim 风格） |
| `Alt+K` | 聚焦上方窗格（vim 风格） |
| `Alt+L` | 聚焦右侧窗格（vim 风格） |

- **点击窗格**：切换焦点到该窗格
- **鼠标滚轮悬停窗格**：滚动该窗格的内容

### 窗格操作

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+X` | 交换当前窗格与下一个窗格的内容 |
| `Ctrl+Shift+Z` | 切换窗格缩放（最大化/恢复） |
| `Ctrl+Shift+Alt+方向键` | 调整分屏比例 |
| `Ctrl+Shift+Alt+B` | 平衡分屏窗格（均匀间距） |
| `Ctrl+Shift+Alt+N` | 重置布局为单个窗格 |

### 窗格缩放

`Ctrl+Shift+Z` 切换缩放模式：
- 缩放时：当前窗格填满整个窗口
- 窗格边框隐藏
- 鼠标焦点锁定在当前窗格
- 分隔符拖拽被禁用
- 状态栏显示 `ZOOM` 指示器

### 分隔符拖拽

- 拖动窗格间的分隔符来调整大小
- 缩放模式下禁用分隔符拖拽

### 多窗格渲染

- 每个窗格独立渲染自己的终端网格
- 当前窗格有明亮的蓝色边框
- 非当前窗格有暗淡的边框
- 窗格间距：窗格之间 6px
- 裁剪矩形确保内容不会跨越窗格边界
