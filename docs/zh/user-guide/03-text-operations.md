# 第三部分：文本选择、复制与粘贴

## 文本选择

### 选择模式

| 操作 | 结果 |
|--------|--------|
| 点击 + 拖拽 | 普通文本选择 |
| `Alt` + 点击 + 拖拽 | 块（矩形）选择 |
| 双击 | 选择单词 |
| 三击 | 选择整行 |

### 键盘选择

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+A` | 全选文本 |
| `Shift+方向键` | 按字符扩展选区 |
| `Ctrl+Shift+Left/Right` | 按单词扩展选区 |

### 选区高亮

选中的文本以半透明蓝色覆盖层高亮显示。块选择模式下，逐行渲染矩形。

### 选区字数统计

选中文本时，状态栏显示字符数和单词数：
- `SEL:42c/7w` — 42 个字符，7 个单词
- 0 个单词时省略 `w` 后缀（纯空白选区）

## 复制

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+C` | 复制选区到剪贴板 |
| `Ctrl+Insert` | 复制选区（Linux/Windows 惯例） |
| `Ctrl+Shift+Alt+H` | 复制选区为 HTML（带颜色） |
| `Ctrl+Shift+Alt+O` | 复制上一条命令的输出（使用 OSC 133 标记） |
| `Ctrl+Shift+Alt+P` | 复制当前工作目录路径 |

通过命令面板的额外复制命令：
- **复制可见文本** — 仅复制可见屏幕内容（不含滚动历史）
- **复制为 Markdown** — 将终端输出转换为 Markdown 格式
- **复制为 HTML** — 保留颜色和格式

### 智能复制裁剪

复制文本时会自动去除首尾空行。

## 粘贴

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+V` | 从剪贴板粘贴 |
| `Shift+Insert` | 粘贴（Linux/Windows 惯例） |
| 中键点击 | 粘贴选区（X11 风格） |

### 括号粘贴

当 shell 支持括号粘贴时（大多数现代 shell 都支持）：
- 粘贴内容用 `ESC[200~ ... ESC[201~` 包裹
- Shell 可以安全处理多行粘贴

### 安全粘贴

当 shell 不支持括号粘贴时：
- 去除末尾换行符以防止意外执行命令
- Toast 通知："Pasted first line (N lines stripped)"

## 搜索

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+F` | 切换浮动搜索栏 |
| `Enter` | 下一个匹配 |
| `Shift+Enter` | 上一个匹配 |
| `Tab`（搜索中） | 切换大小写敏感 |
| `Shift+Tab`（搜索中） | 切换正则搜索模式 |
| `Up/Down`（搜索中） | 浏览搜索历史 |
| `Esc` | 关闭搜索栏 |

### 搜索功能

- **浮动搜索栏** 带匹配计数器（如 "3/12 matches"）
- **搜索历史**：保存最近 20 条查询，可用 Up/Down 浏览
- **大小写切换**：区分或不区分大小写
- **正则模式**：完整正则表达式支持
- **高亮**：匹配项在终端网格中高亮显示

通过命令面板的额外搜索命令：
- **搜索选区** — 使用当前选中文本进行搜索
- **网络搜索** — 在浏览器中搜索选中文本

## 导出

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+Alt+S` | 保存滚动历史到文本文件（`~/ggterm-export-{timestamp}.txt`） |
| `Ctrl+Shift+Alt+E` | 导出终端为 HTML（带颜色） |

通过命令面板的导出命令：
- **导出滚动历史** — 纯文本文件
- **导出为 HTML** — 保留颜色、格式、超链接

## 终端锁定

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+Alt+L` | 切换终端锁定（只读模式） |

锁定后，所有键盘输入将被阻止。适用于阅读输出时防止意外输入。

## 滚动

### 键盘滚动

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+Space` | 切换滚动历史浏览模式（less 风格：j/k/G/g/d/u/q） |
| `Shift+PageUp` | 向上滚动一页 |
| `Shift+PageDown` | 向下滚动一页 |
| `Shift+Home` | 滚动到滚动历史顶部 |
| `Shift+End` | 滚动到底部 |
| `Ctrl+Shift+End` | 滚动到底部（替代方式） |
| `Ctrl+Shift+Alt+Up` | 滚动到标记（OSC 1337 SetMark） |

### 鼠标滚动

- **滚轮**：滚动滚动历史
- **Shift+滚轮**：同步滚动所有窗格
- **滚动条**：点击或拖动右侧边缘的细滚动条

### 滚动指示器

- **滚动到底部药丸**：向上滚动时显示带向下箭头的蓝色药丸，点击跳到底部
- **百分比指示器**：以百分比显示滚动位置（如 "45%"）
- **行数计数**：滚动超过 99 行时显示行数（如 "127 lines"）

### 平滑惯性滚动

支持触控板动量滚动，使用指数衰减插值实现平滑滚动。

### 备用屏幕滚动

在全屏应用（vim、less、htop）且未启用鼠标跟踪时：
- 鼠标滚轮自动转换为方向键按键（DECSET 7727）
- Shift+滚轮绕过此行为并滚动视口
