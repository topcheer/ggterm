# 第五部分：AI 助手

## AI 功能

GGTerm 集成了 AI 助手，可以解释终端输出、建议命令，以及将自然语言转换为 shell 命令。

### AI 快捷键

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+E` | 解释当前输出 |
| `Ctrl+Shift+S` | 建议下一条命令 |
| `Ctrl+Shift+H` | 帮助 — 询问任何关于终端的问题 |
| `Ctrl+Shift+N` | 自然语言转命令 |
| `Esc` | 关闭 AI 覆盖层 |
| `Tab`（AI 覆盖层中） | 将建议命令插入终端 |
| `Ctrl+Enter`（AI 覆盖层中） | 立即执行建议命令 |

### AI 配置

```toml
[ai]
enabled = true
api_endpoint = "https://api.openai.com/v1"
model = "gpt-4"
```

AI 引擎使用兼容 OpenAI 的 API 客户端，因此任何兼容的端点均可使用。

### AI 上下文

触发 AI 功能时，GGTerm 从以下内容构建上下文：
- 当前终端输出（可见屏幕）
- 当前工作目录（OSC 7）
- 运行中的命令（OSC 133 标记）
- 退出码（OSC 133 D 标记）

### 命令面板 AI 命令

通过命令面板（`Ctrl+Shift+P`）：
- `ai.explain` — 解释输出
- `ai.suggest` — 建议命令
- `ai.help` — 通用帮助

## Shell 切换器

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+L` | 快速 Shell 切换器 |

打开下拉菜单，在当前窗格中切换已安装的 shell（bash、zsh、fish 等）。

## 命令历史侧边栏

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+Y` | 切换命令历史侧边栏 |

功能：
- 当前会话中运行的完整命令历史
- 从 OSC 133 标记同步
- 显示退出码（绿色/红色指示器）
- 点击命令可重新运行

## 命令导航

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+Up/Down` | 在命令块之间导航 |

使用 OSC 133 标记在之前的命令输出块之间跳转。

## 代码片段

代码片段管理器存储常用命令：
- 通过命令面板进行增删改查
- TOML 持久化
- 占位符填充（如 `$USER`、`$HOST`）
- 通过命令面板访问

## 广播输入

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+Alt+B` | 循环切换广播模式 |

三种模式：
1. **None** — 输入仅发送到当前窗格（默认）
2. **AllPanes** — 输入发送到当前标签页的所有窗格
3. **AllTabs** — 输入发送到所有标签页的当前窗格

激活时状态栏显示 `BCAST:AllPanes` 或 `BCAST:AllTabs`。

通过命令面板的额外广播命令：
- 向所有窗格发送 `Ctrl+C`
- 重置所有终端

## 会话录制

以 asciinema v2 格式录制终端会话：
- 通过命令面板开始/停止
- 状态栏显示 `REC` 指示器
- 输出：`.cast` 文件

## 工作区

工作区用于分隔标签页组：

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+Alt+W` | 循环切换工作区 |

通过命令面板：
- `workspace.next` — 切换到下一个工作区
- `workspace.prev` — 切换到上一个工作区
- `workspace.add` — 创建新工作区

## 性能监视器

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+G` | 切换性能监视器 |

显示 FPS、内存、单元格计数和 PID 信息的覆盖层。

## 声音

| 快捷键 | 功能 |
|----------|--------|
| `Ctrl+Shift+M` | 切换声音 |

启用后，终端 bell（`\a`）会播放声音。状态栏显示 `SND` 指示器。

## 文件预览

将文件拖拽到终端上方时，显示预览卡片：
- 文件图标（按类别：代码、图片、归档等）
- 文件名和大小
- 类别特定颜色

## 取色器

悬停在 ANSI 颜色序列上时，显示颜色色板及十六进制值。

## 上下文菜单

在终端内容区域右键点击：

| 操作 | 说明 |
|--------|-------------|
| 复制 | 复制选区 |
| 粘贴 | 从剪贴板粘贴 |
| 全选 | 选择所有文本 |
| 搜索 | 打开搜索栏 |
| 清屏 | 清除屏幕 + 滚动历史 |
| 重置 | 重置终端（RIS） |
| 水平分屏 | 左右分屏 |
| 垂直分屏 | 上下分屏 |

## 通知

- **桌面通知**：OSC 9（iTerm2）和 OSC 777（urxvt）协议
- **进度报告**：OSC 9;4 在状态栏显示百分比
- **Bell**：视觉闪烁 + 可选声音
