# Part 4: テーマ、フォントと外観

## テーマ

### 9つのビルトインテーマ + Auto

| テーマ | 背景 | スタイル |
|-------|-----------|-------|
| `dark` | ダークグレー | デフォルトダークテーマ |
| `light` | 白 | 明るい環境向け |
| `dracula` | ダークパープル | 人気のダークテーマ |
| `solarized-dark` | 深い青 | 開発者集中用 |
| `solarized-light` | ウォームクリーム | 読書用 |
| `gruvbox` | アースダーク | レトロウォーム |
| `nord` | アークティックブルー | クリーンミニマル |
| `tokyo-night` | 深いネイビー | ナイトコーディング |
| `catppuccin-mocha` | ソフトブラウン | ウォームダーク |
| `auto` | OS に追従 | シームレスな切り替え |

### テーマ操作

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+Alt+T` | テーマを順に切り替え |
| `Ctrl+Shift+T` | テーマを順に切り替え（代替） |

### Auto テーマ

`theme = "auto"` の場合、GGTerm は OS の外観を検出します：
- **macOS**: AppleInterfaceStyle
- **Linux**: GTK_THEME / gsettings
- **Windows**: レジストリチェック

### ダイナミックカラー（OSC 10/11/12）

プログラムは実行時にテーマカラーをオーバーライドできます：
- `OSC 10` — 前景色の設定/照会
- `OSC 11` — 背景色の設定/照会
- `OSC 12` — カーソル色の設定/照会
- `OSC 104/110/111/112` — デフォルトにリセット

### カスタムパレット（OSC 4）

base16-shell、wal、pywal などのプログラムはカスタム16色パレットを設定できます：
- `OSC 4 ; N ; rgb:RR/GG/BB` — パレットカラー N を設定
- `OSC 104 ; N` — パレットカラー N をリセット
- レンダラーはインデックスカラーにオーバーライドを適用します

## フォント

### フォント操作

| ショートカット | アクション |
|----------|--------|
| `Ctrl+=` | ズームイン（フォントサイズ +1.5px） |
| `Ctrl+-` | ズームアウト（フォントサイズ -1.5px） |
| `Ctrl+0` | デフォルトのフォントサイズにリセット |
| `Ctrl+Shift+Wheel` | マウスホイールでフォントズーム |

### プラットフォーム別デフォルトフォント

| プラットフォーム | フォント |
|----------|------|
| macOS | Menlo（Regular のみ — Bold にはボックス描画グリフがありません） |
| Linux | DejaVu Sans Mono |
| Windows | Cascadia Mono |

**ボールドテキスト**: 明るい色で区別され、フォントウェイトではありません（xterm/Alacritty 標準）。

**CJK フォールバック**: `Shaping::Advanced` により CJK 文字の自動フォントフォールバックが有効になります。

### セル寸法

- セル幅 = 'M' の正確な浮動小数点アドバンス（丸めなし）
- セル高 = ピクセル単位のフォントサイズ
- 実際のピクセル寸法は CSI 14t/15t/16t で報告されます

## 背景の不透明度

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+Alt+]` | 不透明度を上げる（+5%） |
| `Ctrl+Shift+Alt+[` | 不透明度を下げる（-5%） |

不透明度の範囲：0.0（完全に透明）～ 1.0（完全に不透明）。トースト通知でパーセンテージが表示されます。

設定：`[appearance] background_opacity = 0.85`

## ウィンドウ操作

| ショートカット | アクション |
|----------|--------|
| `F11` | フルスクリーンの切り替え |
| `Ctrl+Shift+Enter` | 最大化の切り替え |
| `Ctrl+Shift+Alt+A` | 常に最前面の切り替え |
| `Ctrl+Shift+B` | ステータスバーの切り替え |

### 透明なタイトルバー（macOS）

macOS では、シームレスな外観のためにタイトルバーが透明になります。

## カーソル

### カーソルスタイル

設定：`[appearance] cursor_style = "block"`

オプション：`block`、`underline`、`bar`

プログラムは DECSCUSR（CSI N q）でカーソルスタイルを変更できます。

### カーソルブリンク

設定：`[appearance] cursor_blink = true`

- ブリンクはスムーズなフェードのためにサイン波アルファを使用します
- ユーザー入力時にブリンクがリセットされます
- ブリンクフェーズは SGR 5 ブリンクテキストレンダリングと共有されます

### カーソル行ハイライト

設定：`[appearance] cursor_line_highlight = false`

カーソルが位置する行全体をハイライトします（Vim の `cursorline` のように）。

### カーソルエフェクト

Command Palette から：
- **cursor.trail** — カーソルがパーティクルトレイルを残します
- **cursor.glow** — カーソルにグロー効果を追加します
- **cursor.none** — カーソルエフェクトを無効にします

## ステータスバー

切り替え：`Ctrl+Shift+B`

ステータスバーの表示内容：
- カーソル位置（行:列）
- タブ数
- 現在のディレクトリ（OSC 7 より）
- リモートホスト（OSC 1337 より SSH インジケーター）
- 実行中のコマンド + タイマー
- 進捗パーセンテージ（OSC 9;4 より）
- ブロードキャストモードインジケーター
- 録画インジケーター
- ペインズームインジケーター
- ベルインジケーター
- サウンド切り替えインジケーター
- 選択文字数カウント
- 設定エラーインジケーター

## プロファイル

プロファイルにより、外観設定を切り替えることができます：

```toml
[profiles.develop]
theme = "nord"
font_size = 12

[profiles.present]
theme = "light"
font_size = 18
```

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+Alt+F` | 設定プロファイルを順に切り替え |
| `Ctrl+Shift+Alt+P` | プロファイルを順に切り替え（代替） |

## 設定パネル

| ショートカット | アクション |
|----------|--------|
| `Ctrl+,` | 設定パネルを開く |

矢印キーで設定をナビゲートし、インラインで値を編集します。

## デバッグオーバーレイ

| ショートカット | アクション |
|----------|--------|
| `F1` | デバッグオーバーレイの切り替え（FPS、セル数、ペイン情報） |
| `Ctrl+Shift+G` | パフォーマンスモニターの切り替え |

## ペインごとのレンダリング

各ペインは独立したレンダラー状態を維持します：
- リバースビデオモード（DECSCNM）
- ダイナミック前景/背景色（OSC 10/11）
- 下線色（SGR 58）
- ブリンクテキストフェーズ（SGR 5）
