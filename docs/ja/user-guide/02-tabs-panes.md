# Part 2: タブ、ペインとスプリット

## タブ

### タブ管理

| ショートカット | アクション |
|----------|--------|
| `Ctrl+T` | 新しいタブを開く |
| `Ctrl+W` | 現在のタブを閉じる |
| `Alt+1-9` | タブ N に切り替え |
| `Ctrl+Tab` | 次のタブ |
| `Ctrl+Shift+Tab` | 前のタブ |
| `Ctrl+Shift+\`` | 最後のタブに切り替え（直近の2つを切り替え） |
| `Ctrl+Shift+T` | 最後に閉じたタブを再び開く |
| `Ctrl+Shift+N` | 新しいウィンドウを開く |
| `Ctrl+Shift+I` | 現在のタブの名前を変更 |
| `Ctrl+Shift+PageUp` | タブを左に移動 |
| `Ctrl+Shift+PageDown` | タブを右に移動 |
| `Ctrl+Shift+Alt+D` | 現在のタブを複製（同じシェル + cwd） |
| `Ctrl+Shift+Alt+W` | 他のすべてのタブを閉じる |

### タブの操作

- **タブをクリック**: そのタブに切り替え
- **タブをダブルクリック**: 名前を変更
- **タブを中クリック**: 閉じる（ブラウザスタイル）
- **タブをドラッグ**: 兄弟タブ間で並べ替え
- **タブを右クリック**: コンテキストメニュー（閉じる、他を閉じる、右側を閉じる、ピン留め/解除、スプリット）
- **「+」をクリック**: ドロップダウンメニュー（新しいタブ、水平スプリット、垂直スプリット）

### タブのピン留め

Command Palette からタブをピン留めすると、誤ったクローズを防げます：
- ピン留めされたタブにはピンインジケーターが表示されます
- ピン留めされたタブでは `Ctrl+W` が無視されます
- ピン留めを解除するには Command Palette を使用します

### タブタイトルの同期

タブタイトルは実行中のプログラムと自動的に同期します：
- OSC 0/2 からプログラム名を表示（例：「vim」、「htop」、「less」）
- シェル名にフォールバック（例：「zsh」、「bash」）
- バックグラウンドタブがベルを受信するとベルインジケーターを表示
- alternate screen モード時は `(alt)` を表示

## スプリットペイン

### スプリットの作成

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+D` | 水平スプリット（左 | 右） |
| `Ctrl+Shift+\` | 垂直スプリット（上 / 下） |

新しいペインはアクティブペインの作業ディレクトリを継承します（OSC 7 より）。

### ペインのナビゲーション

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+[` | 前のペインにフォーカス |
| `Ctrl+Shift+]` | 次のペインにフォーカス |
| `Alt+H` | 左のペインにフォーカス（vim スタイル） |
| `Alt+J` | 下のペインにフォーカス（vim スタイル） |
| `Alt+K` | 上のペインにフォーカス（vim スタイル） |
| `Alt+L` | 右のペインにフォーカス（vim スタイル） |

- **ペインをクリック**: そのペインにフォーカスを切り替え
- **ペイン上でマウスホイール**: そのペインのコンテンツをスクロール

### ペインの操作

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+X` | アクティブペインのコンテンツを次と交換 |
| `Ctrl+Shift+Z` | ペインズームの切り替え（最大化/復元） |
| `Ctrl+Shift+Alt+Arrows` | スプリット比率を調整 |
| `Ctrl+Shift+Alt+B` | スプリットペインを均等化（等間隔配置） |
| `Ctrl+Shift+Alt+N` | レイアウトをシングルペインにリセット |

### ペインズーム

`Ctrl+Shift+Z` でズームモードを切り替えます：
- ズーム時：アクティブペインがウィンドウ全体に表示されます
- ペインの境界線は非表示になります
- マウスフォーカスはアクティブペインに固定されます
- セパレータのドラッグは無効になります
- ステータスバーに `ZOOM` インジケーターが表示されます

### セパレータのドラッグ

- ペイン間のセパレータをドラッグしてサイズを変更できます
- ズーム時はセパレータのドラッグが無効になります

### マルチペインレンダリング

- 各ペインは独立してターミナルグリッドをレンダリングします
- アクティブペインには明るい青の境界線があります
- 非アクティブペインには薄い境界線があります
- ペイン間のギャップ：6px
- シザー_RECTによりコンテンツがペイン境界を越えて漏れるのを防ぎます
