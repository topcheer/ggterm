# Part 3: テキスト選択、コピー＆ペースト

## テキスト選択

### 選択モード

| アクション | 結果 |
|--------|--------|
| クリック + ドラッグ | 通常のテキスト選択 |
| `Alt` + クリック + ドラッグ | ブロック（矩形）選択 |
| ダブルクリック | 単語を選択 |
| トリプルクリック | 行全体を選択 |

### キーボードによる選択

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+A` | すべてのテキストを選択 |
| `Shift+Arrows` | 文字単位で選択を拡張 |
| `Ctrl+Shift+Left/Right` | 単語単位で選択を拡張 |

### 選択ハイライト

選択されたテキストは半透明の青いオーバーレイでハイライトされます。ブロック選択の場合、行ごとに矩形がレンダリングされます。

### 選択文字数カウント

テキストが選択されている場合、ステータスバーに文字数と単語数が表示されます：
- `SEL:42c/7w` — 42文字、7単語
- 単語が 0 の場合 `w` サフィックスは省略されます（空白のみの選択）

## コピー

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+C` | 選択範囲をクリップボードにコピー |
| `Ctrl+Insert` | 選択範囲をコピー（Linux/Windows の慣例） |
| `Ctrl+Shift+Alt+H` | 選択範囲を HTML としてコピー（色付き） |
| `Ctrl+Shift+Alt+O` | 最後のコマンド出力をコピー（OSC 133 マークを使用） |
| `Ctrl+Shift+Alt+P` | 現在の作業ディレクトリのパスをコピー |

Command Palette から追加のコピーコマンド：
- **表示テキストをコピー** — 可視画面のみコピー（スクロールバックなし）
- **Markdown としてコピー** — ターミナル出力を Markdown 形式に変換
- **HTML としてコピー** — 色とフォーマットを保持

### スマートコピーのトリミング

コピーされるテキストの先頭と末尾の空行は自動的に削除されます。

## ペースト

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+V` | クリップボードからペースト |
| `Shift+Insert` | ペースト（Linux/Windows の慣例） |
| 中クリック | 選択範囲をペースト（X11 スタイル） |

### Bracketed Paste

シェルが bracketed paste をサポートしている場合（ほとんどのモダンシェルが対応）：
- ペーストされるコンテンツは `ESC[200~ ... ESC[201~` でラップされます
- シェルは複数行のペーストを安全に処理できます

### セーフペースト

Bracketed paste がサポートされていない場合：
- 末尾の改行は削除され、意図しないコマンド実行を防ぎます
- トースト通知：「Pasted first line (N lines stripped)」

## 検索

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+F` | フローティング検索バーの切り替え |
| `Enter` | 次の一致 |
| `Shift+Enter` | 前の一致 |
| `Tab`（検索中） | 大文字小文字の区別の切り替え |
| `Shift+Tab`（検索中） | 正規表現検索モードの切り替え |
| `Up/Down`（検索中） | 検索履歴のナビゲート |
| `Esc` | 検索バーを閉じる |

### 検索機能

- **フローティング検索バー** 一致カウンター付き（例：「3/12 matches」）
- **検索履歴**: 過去20件のクエリを保存、Up/Down でナビゲート
- **大文字小文字の切り替え**: 大文字小文字を区別する/しない
- **正規表現モード**: 完全な正規表現サポート
- **ハイライト**: ターミナルグリッド内で一致をハイライト

Command Palette からの追加検索コマンド：
- **選択テキストを検索** — 現在選択されているテキストで検索
- **Web で検索** — 選択したテキストをブラウザで検索

## エクスポート

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+Alt+S` | スクロールバックをテキストファイルに保存（`~/ggterm-export-{timestamp}.txt`） |
| `Ctrl+Shift+Alt+E` | ターミナルを HTML としてエクスポート（色付き） |

Command Palette からのエクスポートコマンド：
- **スクロールバックをエクスポート** — プレーンテキストファイル
- **HTML としてエクスポート** — 色、フォーマット、ハイパーリンクを保持

## ターミナルロック

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+Alt+L` | ターミナルロックの切り替え（読み取り専用モード） |

ロック中、すべてのキーボード入力がブロックされます。誤って入力することなく出力を読みたい場合に便利です。

## スクロール

### キーボードスクロール

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+Space` | スクロールバックブラウズモードの切り替え（less スタイル: j/k/G/g/d/u/q） |
| `Shift+PageUp` | 1ページ上にスクロール |
| `Shift+PageDown` | 1ページ下にスクロール |
| `Shift+Home` | スクロールバックの先頭へスクロール |
| `Shift+End` | 最下部へスクロール |
| `Ctrl+Shift+End` | 最下部へスクロール（代替） |
| `Ctrl+Shift+Alt+Up` | マークへスクロール（OSC 1337 SetMark） |

### マウススクロール

- **スクロールホイール**: スクロールバック履歴をスクロール
- **Shift+スクロール**: すべてのペインを同時にスクロール
- **スクロールバー**: 右端の細いスクロールバーをクリックまたはドラッグ

### スクロールインジケーター

- **最下部へスクロールピル**: スクロールアップ時に下矢印付きの青いピルが表示、クリックで最下部にジャンプ
- **パーセンテージインジケーター**: スクロール位置をパーセンテージで表示（例：「45%」）
- **行数**: 99行以上スクロール時に行数を表示（例：「127 lines」）

### スムーズ慣性スクロール

トラックパッドの慣性に対応し、指数関数的減衰補間でスムーズなスクロールを実現します。

### Alternate Screen Scroll

マウストラッキングなしのフルスクリーンアプリ（vim、less、htop）内では：
- マウスホイールが自動的に矢印キー入力に変換されます（DECSET 7727）
- Shift+ホイールはこれをバイパスし、ビューポートをスクロールします

## ターミナルロック（読み取り専用）

`Ctrl+Shift+Alt+L` でロック状態を切り替えます：
- ロック中、すべてのキーボード入力が無視されます
- 出力を読む際に誤ってタイプするのを防ぎます

## ファイルプレビュー

ターミナル上にファイルをドラッグすると、プレビューカードが表示されます：
- ファイルアイコン（カテゴリ別：コード、画像、アーカイブなど）
- ファイル名とサイズ
- カテゴリ固有の色

## カラーピッカー

ANSI カラーシーケンス上にカーソルを合わせると、HEX 値を表示するカラースウォッチが表示されます。

## コンテキストメニュー

ターミナルコンテンツエリアで右クリック：

| アクション | 説明 |
|--------|-------------|
| Copy | 選択範囲をコピー |
| Paste | クリップボードからペースト |
| Select All | すべてのテキストを選択 |
| Search | 検索バーを開く |
| Clear | 画面 + スクロールバックをクリア |
| Reset | ターミナルをリセット（RIS） |
| Split Horizontal | 左右にスプリット |
| Split Vertical | 上下にスプリット |

## 通知

- **デスクトップ通知**: OSC 9（iTerm2）および OSC 777（urxvt）プロトコル
- **進捗報告**: OSC 9;4 がステータスバーにパーセンテージを表示
- **ベル**: ビジュアルフラッシュ + オプションのサウンド
