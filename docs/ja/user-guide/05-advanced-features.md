# Part 5: AI アシスタント

## AI 機能

GGTerm はターミナル出力の説明、コマンドの提案、自然言語からシェルコマンドへの変換ができる AI アシスタントを統合しています。

### AI ショートカット

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+E` | 現在の出力を説明 |
| `Ctrl+Shift+S` | 次のコマンドを提案 |
| `Ctrl+Shift+H` | ヘルプ — ターミナルについて何でも質問 |
| `Ctrl+Shift+N` | 自然言語からコマンドへ変換 |
| `Esc` | AI オーバーレイを閉じる |
| `Tab`（AI オーバーレイ中） | 提案コマンドをターミナルに挿入 |
| `Ctrl+Enter`（AI オーバーレイ中） | 提案コマンドを即座に実行 |

### AI 設定

```toml
[ai]
enabled = true
api_endpoint = "https://api.openai.com/v1"
model = "gpt-4"
```

AI エンジンは OpenAI 互換の API クライアントを使用するため、互換性のあるエンドポイントならどれでも動作します。

### AI コンテキスト

AI 機能の起動時、GGTerm は以下からコンテキストを構築します：
- 現在のターミナル出力（可視画面）
- 現在の作業ディレクトリ（OSC 7）
- 実行中のコマンド（OSC 133 マーク）
- 終了コード（OSC 133 D マーク）

### Command Palette の AI コマンド

Command Palette（`Ctrl+Shift+P`）から：
- `ai.explain` — 出力を説明
- `ai.suggest` — コマンドを提案
- `ai.help` — 一般的なヘルプ

## シェルスイッチャー

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+L` | クイックシェルスイッチャー |

インストールされているシェル（bash、zsh、fish など）を現在のペインで切り替えるドロップダウンを開きます。

## コマンド履歴サイドバー

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+Y` | コマンド履歴サイドバーの切り替え |

機能：
- 現在のセッションで実行したコマンドの全履歴
- OSC 133 マークから同期
- 終了コードを表示（緑/赤インジケーター）
- コマンドをクリックして再実行

## コマンドナビゲーション

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+Up/Down` | コマンドブロック間をナビゲート |

OSC 133 マークを使用して、前のコマンド出力ブロック間をジャンプします。

## スニペット

スニペットマネージャーはよく使うコマンドを保存します：
- Command Palette から CRUD 操作
- TOML 永続化
- プレースホルダー補完（例：`$USER`、`$HOST`）
- Command Palette からアクセス

## ブロードキャスト入力

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+Alt+B` | ブロードキャストモードの切り替え |

3つのモード：
1. **None** — 入力はアクティブペインのみ（デフォルト）
2. **AllPanes** — アクティブタブ内のすべてのペインに入力を送信
3. **AllTabs** — すべてのタブのアクティブペインに入力を送信

アクティブ時、ステータスバーに `BCAST:AllPanes` または `BCAST:AllTabs` が表示されます。

Command Palette からの追加ブロードキャストコマンド：
- すべてのペインに `Ctrl+C` を送信
- すべてのターミナルをリセット

## セッション録画

ターミナルセッションを asciinema v2 形式で録画します：
- Command Palette から開始/停止
- ステータスバーに `REC` インジケーターを表示
- 出力：`.cast` ファイル

## ワークスペース

ワークスペースはタブのグループを分離します：

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+Alt+W` | ワークスペースを切り替え |

Command Palette から：
- `workspace.next` — 次のワークスペースへ切り替え
- `workspace.prev` — 前のワークスペースへ切り替え
- `workspace.add` — 新しいワークスペースを作成

## パフォーマンスモニター

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+G` | パフォーマンスモニターの切り替え |

FPS、メモリ、セル数、PID 情報を含むオーバーレイを表示します。

## サウンド

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+M` | サウンドの切り替え |

有効時、ターミナルベル（`\a`）が音声を再生します。ステータスバーに `SND` インジケーターが表示されます。

## ファイルプレビュー

ターミナル上にファイルをドラッグすると、プレビューカードが表示されます：
- ファイルアイコン（カテゴリ別：コード、画像、アーカイブなど）
- ファイル名とサイズ
- カテゴリ固有の色

## カラーピッカー

ANSI カラーシーケンス上にカーソルを合わせると、HEX 値を表示するカラースウォッチが表示されます。

## コンテキストメニュー

ターミナルコンテンツエリアで右クリック：

| アクション | 説明 |
|--------|-------------|
| Copy | 選択範囲をコピー |
| Paste | クリップボードからペースト |
| Select All | すべてのテキストを選択 |
| Search | 検索バーを開く |
| Clear | 画面 + スクロールバックをクリア |
| Reset | ターミナルをリセット（RIS） |
| Split Horizontal | 左右にスプリット |
| Split Vertical | 上下にスプリット |
